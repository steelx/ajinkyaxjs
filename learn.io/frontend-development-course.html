Frontend Web Development Course Syllabus

Foundation

During our Frontend Web Development course, students gain the knowledge required to build frontend web applications from scratch. Learn common developer tools such as Git, GitHub, the command line, and Brackets, our recommended text editor. Explore HTML and CSS, including responsive web development. Study fundamental programming concepts using JavaScript, and apply those concepts while building real applications.

Learn tools like Git, GitHub, and the command line
Explore HTML, CSS, and responsive web development

Projects

In the Project Phase of your apprenticeship, you'll leverage the fundamentals learned in the Foundation Phase to build three to five production quality websites. You'll work on a series of applications to create a portfolio that showcases your talent. During the Project Phase, your mentor will be your client, senior developer, and project manager.
